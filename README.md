# Baby.Standen.link

This website is for crowd sourcing names and voting on them.

## Development

### Start

Single command

```sh
yarn run dev
```

Otherwise with *watch* enabled, use two terminals

```sh
yarn run watch
cd client && yarn start
```

### Test

```sh
yarn test
```

### Deploy

```sh
git push heroku master
```
